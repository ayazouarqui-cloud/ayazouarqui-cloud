### 🚀 Projets
 
**🔵 Détection & prédiction de congestion réseau LTE — PFE Djezzy (Optimum Telecom Algérie)**
Pipeline complet sur 7,78M enregistrements horaires / 72 851 cellules : EDA, feature engineering, labellisation non supervisée (ACP + K-Means), benchmark de 6 modèles ML (**LightGBM retenu, F1 = 99,6 %**), prédiction multi-horizon H+1/H+3/H+6 (jusqu'à 98 % à H+6), interprétabilité SHAP, optimisation Optuna, déploiement Streamlit.
📌 *Repo en cours de publication.*
 
**📊 [Pipeline ETL — Power BI (Northwind)](https://github.com/ayazouarqui-cloud/PowerBI-Northwind-ETL-Visualization)**
Nettoyage, consolidation et automatisation d'un pipeline ETL multi-sources (SQL Server, Access) pour le reporting décisionnel. Modélisation en étoile, mesures DAX, visualisations Python intégrées.
 
**🚕 Streaming temps réel — Taxis (3M+ lignes)**
Traitement de flux en temps réel avec Kafka, MongoDB et Cassandra, architecture déployée via Docker.
📌 *Repo en cours de publication.*
